# Automation Testing Practice 🚀

Este repositorio contiene los ejercicios y proyectos prácticos que he realizado para mi formación en **QA Automation**. El objetivo es documentar mi progreso y organizar el código generado durante las clases.

## 🎯 Competencias Desarrolladas

Al finalizar este proceso, demuestro dominio en las siguientes áreas técnicas:

*   **Lenguaje Principal:** Desarrollo de scripts de automatización utilizando **Python**.
*   **Web Automation:** Automatización de interfaces web con **Selenium WebDriver**.
*   **Frameworks de Testing:** Implementación de pruebas con **Pytest** y enfoque **BDD (Behavior Driven Development)** utilizando **Behave**.
*   **Patrones de Diseño:** Creación de frameworks mantenibles y escalables mediante el patrón **Page Object Model (POM)**.
*   **Data-Driven Testing:** Implementación de pruebas parametrizadas para maximizar la cobertura de escenarios.
*   **API Testing:** Automatización de pruebas para servicios back-end (APIs REST) utilizando la librería **Requests**.
*   **Reportes y Debugging:** Generación de reportes detallados, estrategias de **logging** y captura de errores para una depuración eficiente.
*   **CI/CD:** Integración de pruebas automatizadas en pipelines de integración continua mediante **GitHub Actions**.
